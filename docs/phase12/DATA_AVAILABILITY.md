# Data Availability

Date: July 17, 2026

## Availability Statement

Data and machine-readable evidence used in this repository are available inside the repository and the official Zenodo archive associated with:

`https://doi.org/10.5281/zenodo.20195953`

## Included Data Categories

- tracked configuration files under `config/`
- benchmark-oriented artifacts under `docs/phase6/`
- mission and software HIL artifacts under `docs/phase7/` and `experiments/`
- advanced research scenario outputs under `docs/phase8/`
- AI evaluation artifacts under `docs/phase9/` and `docs/phase11/`
- benchmark and autonomy datasets under `datasets/`

## Access Conditions

- repository-contained artifacts are open within the source tree
- the archived software artifact is available through the official DOI
- no proprietary hardware dataset is required to reproduce the documented software evidence

## Boundaries

- no physical flight dataset is claimed
- no PX4 log archive is claimed
- no Gazebo or SITL dataset is claimed
- no hidden external dataset is required for the documented Phase 12 replication workflow
