# Software Availability

Date: July 17, 2026

## Availability Statement

The software is available as:

- the repository working tree
- the official Zenodo archive at `https://doi.org/10.5281/zenodo.20195953`

## Main Software Components

- native C++ autonomy stack in `src/` and `include/`
- Go control plane in `cmd/` and `internal/`
- Python dashboard and experiment tooling in `gui/` and `scripts/`
- simulation and digital twin code in `simulation/`
- AI and research code in `research/`
- deployment and production-readiness assets in `deployment/`

## Licensing Note

- The repository `LICENSE` file states `GNU GPL v3.0`.
- The DOI metadata retrieved during Phase 12 reported `Creative Commons Attribution 4.0 International` for the Zenodo record metadata.

Researchers using the artifact should preserve both records and verify downstream obligations for their specific use case.

## Availability Boundary

Software availability does not imply:

- hardware shipment
- autopilot firmware release
- PX4 integration evidence
- ROS2 package availability
- field deployment approval
