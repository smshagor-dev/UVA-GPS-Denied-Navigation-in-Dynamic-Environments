# Phase 8 AI Research Interface

Date: July 16, 2026

## Purpose

Phase 8 adds a research-facing interface layer for future AI/ML experiments without inventing model metrics.

## Interface Areas

### Perception Models

- object detection adapters
- semantic classification adapters
- frame-level confidence export

### Localization Models

- learned correction modules
- confidence calibration helpers
- replay or benchmark evaluation hooks

### Prediction Models

- obstacle or trajectory prediction interfaces
- uncertainty export hooks

### Reinforcement Learning Experiments

- policy configuration manifests
- episode metadata
- reward logging schema

### Policy Evaluation

- deterministic scenario scoring
- safety constraint checks
- latency and recovery impact assessment

## Repository Structure

- `research/models/`
- `research/datasets/`
- `research/evaluation/`
- `research/experiments/`

## Non-Claims

- no trained model weights are bundled here
- no accuracy, reward, or generalization numbers are claimed
- no external benchmark leaderboard result is claimed
