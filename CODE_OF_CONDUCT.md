# Code of Conduct

## Our Standard

Contributors should keep collaboration respectful, evidence-based, and constructive.

Expected behavior:

- communicate with professionalism
- focus criticism on code, documentation, and evidence
- document risks honestly
- avoid harassment, hostility, and personal attacks

## Scope

This applies to repository discussions, issues, pull requests, and shared validation artifacts.

## Enforcement

Project maintainers may remove abusive content, limit participation, or reject contributions that violate this standard.
